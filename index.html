üåç Public Access Commands

/server

Fetches a detailed status report of the game server, including player count and server settings.

Usage: /server

/players

Displays a live manifest of all personnel currently active in the server, with team and callsign details.

Usage: /players

/roster

Provides a clear roster of all active personnel, organized by their assigned department/team.

Usage: /roster

/vehicles

Lists all currently deployed vehicles in the server and their registered owners.

Usage: /vehicles

/queue

Shows a list of personnel awaiting entry into the server.

Usage: /queue

/ticket

Opens a secure, private channel to communicate with staff. You must select a category for your report.

Usage: /ticket category:[Choose from list]

/howtosts

Displays the standard operating procedure for conducting a Side-to-Side (STS) event.

Usage: /howtosts

üõ°Ô∏è Restricted Staff Commands

Trainee Level Clearance Trainee+

/requesttraining

Submits a formal request to Command Staff for a training session. Must include your availability.

Usage: /requesttraining availability:Weekends after 5 PM EST

Admin Level Clearance Admin+

/promote

Assigns a Discord role to a user. Cannot assign roles higher than your own.

Usage: /promote user:@User role:@Role

/demote

Removes a Discord role from a user. A reason is required for the log.

Usage: /demote user:@User role:@Role reason:Inactivity

/kick

Removes a player from the game server for a minor infraction.

Usage: /kick player_name:RobloxUser reason:FRP

/ban

Permanently removes a player from the game server for a major infraction.

Usage: /ban player_name:RobloxUser reason:Exploiting

/banlist

Retrieves the complete list of all players banned from the game server.

Usage: /banlist

/whois

Conducts a background check on a player, showing their current status, team, and ban history.

Usage: /whois player_name:RobloxUser

/ticketadmin

Management toolkit for support tickets (close, add/remove user).

Usage: /ticketadmin close

Owner Level Clearance Owner+

/hire

Onboards a new staff member, assigning the Trainee role and sending a welcome notification.

Usage: /hire user:@NewStaff reason:Passed interview

/decline

Formally declines a staff applicant, sending a notification with the reason provided.

Usage: /decline user:@Applicant reason:Not a good fit

/eventcreate

Opens the event creation utility to define or update a step-by-step event sequence.

Usage: /eventcreate

/eventstart

Initiates a pre-saved event, deploying the interactive control panel.

Usage: /eventstart

/announce

Broadcasts a high-priority announcement to the entire game server.

Usage: /announce message:Server restart in 5 mins.

/run

Executes a raw command on the game server. For advanced use only.

Usage: /run command_to_run::m Test message.

/tp

Relocates one player to another player's position in-game.

Usage: /tp player1:UserA player2:UserB

/addadmin & /addmod

Manages bot permissions, promoting users to Admin or Mod.

Usage: /addadmin user:@StaffMember

System Developer Clearance GameDev

/addowner & /removeowner

Manages top-level bot permissions for Owners.

Usage: /addowner user:@HeadAdmin

/sync

Manually forces a re-synchronization of all bot commands with Discord's API.

Usage: /sync
